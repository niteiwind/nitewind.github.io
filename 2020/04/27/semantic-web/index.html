Semantic Web

Semantic Web i6208689

We have walked for long time from Web 1.0 to Web 2.0 and now we entered into a new era – the Age of Web 3.0. Perhaps, people are more willing to name it as ‘Semantic Web’, the web of data. Compared to Web 2.0, the envision of Web 3.0 is to break up the boundaries between different platforms and enable the information or resources could be shared and more accessible to users. The semantic web is not a new thing, the concept of which could be traced to May 2001, Tim Berners-Lee described the informal definition of semantic web in his Scientific American article, that says “The Semantic Web is an extension of the current web in which information is given well-defined meaning, better enabling computers and people to work in cooperation.”

Nowadays we install different applications on devices like phones and computers to meet our diverse demands. For example, we use the self-tracking applications to take over the self-surveillance work: application A for recording how many steps the user walked in a day, and meanwhile application B for calculating how much calories the user took in. what if the user wants to combine all the data from different applications and improve health management? As we all know, data is separately stored by the application which collects it. Therefore, the purpose of Semantic Web is to provide a fundamental system that allows these respectively stored data to be shared, reused, and even interconnected across different applications. The data generated by individuals could constitute a network or web.

Though suggested by the same scientist Tim Berners-Lee, semantic web works differently from the World Wide Web. The WWW we used now is more related to a medium that share, store images, and texts. Each web page consists of HTML documents contain the information which has been transformed into a language that machine could understand (like HTML tags) then rendered on our screen. The human could read the content of the web page, but the machine only knows it is a page. What semantic web desire to achieve is “enabling machines ‘understand’ and respond to complex human requests based on their meaning (Choudhury, p. 8098).” Instead of machines doing AI-based identification, semantic web system provides a clear, machine-readable content within standard semantic description framework (such as Uniform Resources Identifiers, UNICODE, and Resources Description Framework) and build a standard semantic reasoning and trust systems based on the structure. Semantic web could help humans to express the meaning behind the data in a more abundant way that allows the machine to understand the data.

Perhaps one day when you need to see a dentist for a toothache, an intelligent agent will automatically check your schedule, find the right available dentist, automatically check in with the practice’s agent, and arrange your trip for you. However, we still do not know what semantic web would bring to us, and when the world it described would come to us.
